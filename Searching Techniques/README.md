## ğŸ§® Algorithm Basics

### ğŸ”¹ Linear Search
- Checks each element one by one until the key is found (or not found).
- Works on **unsorted data**.

**Time Complexity:** O(n)

### ğŸ”¹ Binary Search
- Works only on **sorted data**.
- Divides the array in half repeatedly to locate the key faster.

**Time Complexity:** O(log n)
